# 🎖️ Military Rank Trainer (C#)

Интерактивный тренажер для заучивания воинских званий и погонов. 

## ✨ Особенности
* **Визуальное обучение:** Отображение картинок погонов в реальном времени.
* **Механика квиза:** Выбор одного из 4 вариантов ответа.
* **Портативность:** Программа работает без установки (запуск через .exe).

## 🛠 Технологии
* **Язык:** C#
* **Платформа:** Windows Forms (.NET Framework)

## 🚀 Как запустить
1. Скачайте файл `WindowsFormsApp1.exe` из этого репозитория.
2. Запустите его на Windows.
*Примечание: Если вы разработчик, откройте `WindowsFormsApp1.sln` в Visual Studio для просмотра кода.*
