 🎖️ Military Rank Trainer (C#)

[Русское описание ниже](#rus)

Interactive desktop trainer for learning military ranks and insignia using a visual quiz system. 

## ✨ Features
* **Visual Learning:** Real-time display of rank insignia images.
* **Quiz Mechanics:** Classic "choose 1 of 4" multiple-choice system.
* **Portable:** Standalone `.exe` file, no installation required.

## 🛠 Tech Stack
* **Language:** C#
* **Platform:** Windows Forms (.NET Framework)

## 🚀 How to Run
1. Download `WindowsFormsApp1.exe` from this repository.
2. Run it on your Windows PC.
*Note: For developers, open `WindowsFormsApp1.sln` in Visual Studio.*
